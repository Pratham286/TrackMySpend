## Technologies Used

- **Frontend**: React.js, JavaScript, CSS

## Features

- Add, edit, and delete expenses
- Categorize expenses
- View total expenses and income

## Usage

1. Add expenses and income transactions.
2. Filter and categorize your transactions.
3. View your financial summary.
